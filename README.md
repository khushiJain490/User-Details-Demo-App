# User-Details-Demo-App

This is a demo app which displays the list of users on login. The components and tool used in this application are
**UI/UX designing
  CardView.  
  Recycler view. 
  Fragments. 
  Activity. 
  AlertDialog box builder. 
   Patterns for validation of phone and email. 
  Adapter. 
  Retrofit. 
  Glide library. 
  Intents. 
  Google Maps API**.
